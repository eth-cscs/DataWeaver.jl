DataWeaver.adios2_config — Function

If the ADIOS2 config file is not provided, this function uses the XML template to create the file automatically with parameters provided by the user. Returns the XML path to the new file.

source
DataWeaver.adios2_init — Method

Intialize ADIOS2. Supports several configuration options: MPI, serial, existing XML config file or a new file with specified parameters.
